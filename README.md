# 绿幕软件 (GreenCurtain)

一个轻量级的 WPF 绿幕软件，支持多显示器、透明度调节、颜色吸取、自定义快捷键等功能。

## 功能特性

- **全屏绿幕覆盖** - 在指定显示器上显示全屏绿幕
- **透明点击** - 绿幕覆盖时不影响鼠标操作
- **多显示器支持** - 可选择在任意显示器上显示绿幕
- **透明度调节** - 支持 10%-100% 的透明度调节（1% 步进）
- **颜色自定义** - 支持自定义绿幕颜色
- **屏幕颜色吸取** - 从屏幕任意位置吸取颜色
- **亮度补偿** - 支持中心和边缘亮度独立调节，补偿屏幕亮度不均匀
- **自定义快捷键** - 支持自定义退出和显示/隐藏快捷键
- **配置保存** - 自动保存和恢复用户设置

## 快捷键

| 快捷键 | 功能 | 默认值 |
|--------|------|--------|
| 退出程序 | 关闭绿幕软件 | `Alt + Q` |
| 显示/隐藏 | 切换绿幕显示状态 | `Alt + A` |

> 快捷键支持 Alt、Ctrl、Shift 修饰键组合，可在设置中自定义

## 使用方法

1. 运行 `GreenCurtain.exe`
2. 程序启动后会在系统托盘显示图标
3. 右键点击托盘图标，选择"设置"打开设置窗口
4. 在设置窗口中可以：
   - 选择目标显示器
   - 调整透明度
   - 更换绿幕颜色（支持颜色选择器和屏幕吸取）
   - 调整中心/边缘亮度补偿
   - 自定义快捷键

## 设置说明

### 透明度
- 范围：10% - 100%
- 步进：1%
- 数值越低，绿幕越透明

### 中心亮度 / 边缘亮度
- 范围：0 - 255
- 默认值：128（原始亮度）
- 用于补偿屏幕四角亮度不均匀的问题

### 快捷键设置
- 点击快捷键输入框
- 按下想要的组合键（如 Ctrl+Shift+Q）
- 设置会自动保存并立即生效
- 支持的修饰键：Alt、Ctrl、Shift

## 系统要求

- Windows 10/11
- .NET 10 Desktop Runtime

## 构建项目

```bash
# 构建
dotnet build

# 运行
dotnet run

# 发布（依赖框架）
dotnet publish -c Release

# 发布（自包含）
dotnet publish -c Release -r win-x64 --self-contained
```

## 配置文件

配置文件保存在：`%AppData%\GreenCurtain\config.json`

保存的设置包括：
- 绿幕颜色（RGB）
- 透明度
- 中心亮度
- 边缘亮度
- 显示器索引
- 退出快捷键
- 显示/隐藏快捷键
